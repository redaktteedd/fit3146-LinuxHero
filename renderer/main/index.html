Essential Commands

ls - List directory contents
Example: ls -la
cd - Change directory
Example: cd /home/user
pwd - Print working directory
Example: pwd
mkdir - Create directory
Example: mkdir new_folder
rm - Remove files/directories
Example: rm -rf folder_name
cp - Copy files
Example: cp file.txt backup.txt
mv - Move/rename files
Example: mv old_name.txt new_name.txt
cat - Display file contents
Example: cat filename.txt

File Permissions

Linux uses a 3-tier permission system:

  • Owner (u) - File owner permissions
  • Group (g) - Group member permissions
  • Others (o) - Everyone else permissions
chmod 755 file.txt - Owner: read/write/execute, Group/Others: read/execute
chmod u+x script.sh - Add execute permission for owner

Navigation Tips

cd ~ - Go to home directory
cd .. - Go up one directory
cd - - Go to previous directory
ls -R - List directories recursively

Process Management

ps - Show running processes
Example: ps aux (show all processes)
top - Real-time process viewer
Interactive process monitor
kill - Terminate processes
Example: kill -9 1234 (force kill process)
jobs - Show background jobs
Example: jobs (list background tasks)

System Information

uname - System information
Example: uname -a (all system info)
df - Disk space usage
Example: df -h (human-readable format)
free - Memory usage
Example: free -h (memory in human format)
uptime - System uptime
Shows how long system has been running

Package Management

APT (Ubuntu/Debian):
sudo apt update - Update package list
sudo apt install package_name - Install package
sudo apt remove package_name - Remove package
YUM/DNF (Red Hat/CentOS):
sudo yum install package_name - Install package
sudo dnf install package_name - Install package (newer)

üêß Shell Scripting

Basic Script Structure

#!/bin/bash
# This is a comment
echo "Hello, World!"

# Variables
name="Linux User"
echo "Welcome, $name"

# User input
read -p "Enter your name: " username
echo "Hello, $username!"

Variables and Control Structures

# Variables
count=10
message="Processing..."

# If statement
if [ $count -gt 5 ]; then
    echo "Count is greater than 5"
else
    echo "Count is 5 or less"
fi

# For loop
for i in {1..5}; do
    echo "Iteration $i"
done

# While loop
while [ $count -gt 0 ]; do
    echo "Count: $count"
    count=$((count - 1))
done

Functions and File Operations

# Function definition
greet_user() {
    local name=$1
    echo "Hello, $name!"
}

# Call function
greet_user "Alice"

# File operations
if [ -f "file.txt" ]; then
    echo "File exists"
    cat file.txt
else
    echo "File not found"
fi

# Directory check
if [ -d "mydir" ]; then
    echo "Directory exists"
    ls -la mydir
fi

Making Scripts Executable

1. Create your script: nano myscript.sh
2. Add shebang: #!/bin/bash
3. Make executable: chmod +x myscript.sh
4. Run script: ./myscript.sh
